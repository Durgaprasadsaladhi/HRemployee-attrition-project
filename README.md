# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition data using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objectives
- Analyze employee attrition trends
- Identify factors affecting attrition
- Perform data visualization

## Files Included
- HR-Employee-Attrition.ipynb
- HR_Employee_Attrition_clean.csv

## Key Insights
- Employees working overtime had higher attrition
- Younger employees showed higher attrition rate
- Job satisfaction impacts employee retention
